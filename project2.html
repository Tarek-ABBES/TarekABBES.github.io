Controlling 6 DOF Robot (UR 10) using Python

Project Description

This project involves robot modeling, motion planning, and implementing a P controller to enable the robot to track a desired trajectory using Python and its NumPy library. and validating these results first through simulation and then by implementation on the real robot using the URX library.

Features:

Python
numpy
Procedural Programming
matplotlib
CoppeliaSim
